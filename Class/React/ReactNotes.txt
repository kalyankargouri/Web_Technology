React ->
    React is a js library developed by meta.
    To build a fast and interactive user interfaces.
    It is a single page application.
    Page does not reload only content change (ex->insta, facebook)

 why do we use react?
    so problem with traditiona js-> 1>full page reload, 
                                    2>complex DOM manipulation, 
                                    3>hard to manage large application

    sol -> update only required parts
           use reusable components


Virtual DOM - 
    react creates copy of real DOM means virtual DOM.
    when data changes react updates virtual DOM.
    updates only change a part in real DOM -> used for high performance.
